# Assignment 01 – Wireshark Display Filters

## Objective

The objective of this assignment was to understand how Wireshark display filters work by analyzing different types of network traffic.

## Filters Tested

- `udp`
- `tls or ssl`

## Files

- `report.pdf` – Assignment report
- `udp-filter.png` – Screenshot of the UDP filter
- `tls-filter.png` – Screenshot of the TLS/SSL filter

## Summary

Using the `udp` filter, I observed protocols such as DNS, NBNS, LLMNR, SSDP, and DHCPv6.

Using the `tls or ssl` filter, I observed the TLS handshake and encrypted HTTPS communication between the client and the server.

## Tools Used

- Wireshark
